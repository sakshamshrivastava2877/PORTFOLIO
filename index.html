SPOTIFY API

I've developed a Spotify API that enhances the user experience by allowing them to search for music within their preferred genre. Utilizing JavaFX for the interface, users can select a genre of their choice, and the API dynamically fetches and displays a curated list of music from that genre. Additionally, it provides detailed descriptions of the songs and tracks the number of views each song has garnered. This interactive and informative platform offers users a seamless way to explore and enjoy music tailored to their tastes.

MEMORY GAME

This memory game challenges players to observe and replicate sequences of flashing boxes. Players must remember the pattern in which the boxes flash and reproduce it accurately. If the player successfully recreates the pattern, they earn points. If they fail, the game ends. The points are recorded under the player's name and saved as a high score on a separate page.
